Sumilla del Módulo

El módulo es de naturaleza teórico-práctico. Le permitirá al participante utilizar gestiones de proyecto de software con un backlog de Jira, y lo integran con GitHub para su implementación y control de versión de código.

Las temáticas que comprenden este módulo son:

  • Introducción a DevOps
  • Gestión y versionamiento de código fuente (GitHub)
  • Gestión de requerimientos de software y Backlog (Jira)
